# Student Hub — UI Kit

The Magnolia Math Suite portal dashboard, **re-skinned in the real Magnolia brand layer** (the repo ships a placeholder brown/gold palette — this is the intended look). Open `index.html`.

**Composition:** a white left sidebar (magnolia logo, Lucide nav with coral active state, "coming soon" locked items, user footer) beside a main column with a blossom-pink gradient greeting hero ("Hi Simon" + next-session card), then "At a glance" stat tiles and session rows. Nav switches panels (Dashboard / Schedule / Boards / Tutor) client-side.

Mirrors the real routes in `magnolia-math-suite/app/portal/*`. Functional UI icons are **Lucide** (the platform's choice); brand doodle icons are reserved for feature markers. Status pills: coral = scheduled/active, green = completed.
