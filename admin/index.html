Connect to GitHub so "Save & Deploy" commits content.js to your repo. Netlify auto-deploys on every commit (~30 seconds).

How it works: When you click "Save & Deploy", this admin reads content.js from your GitHub repo, replaces the content with your edits, and commits the updated file. Netlify detects the new commit and rebuilds your site automatically.

GitHub PAT: You need a Personal Access Token with repo scope. Create one at github.com → Settings → Developer settings → Personal access tokens.
